## **📕 문제 설명**

- 정렬된 정수 배열이 있습니다. 이 배열의 모든 원소들을 오른쪽으로 랜덤하게 n번 이동하였습니다.

- 예를 들면 [1, 2, 3, 4, 5] > [3, 4, 5, 1, 2]

- 이런 배열과 정수 k가 주어지면, 배열안에 k가 존재하는지 찾으시오.

- 존재한다면 배열의 인덱스, 존재하지 않다면 -1을 리턴하시오.

- 시간복잡도 제한 O(log N)

## **📙 입출력 예**

```js
Input: [3, 4, 5, 1, 2], 4
Output: 1

Input: [2, 4, 5, 1], 3
Output: -1

Input: [4, 6, 7, 8, 1, 2, 3], 5
Output: -1
```
