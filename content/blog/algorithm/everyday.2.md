문제 설명
-------

- 문자열 배열(string array)이 주어지면, 제일 긴 공통된 접두사(prefix)의 길이를 구하시오.

입출력 예
-------
```sh

Input: [“apple”, “apps”, “ape”]
Output: 2 // “ap”

Input: [“hawaii”, “happy”]
Output: 2 // “ha”

Input: [“dog”, “dogs”, “doge”]
Output: 3 // “dog”

```
