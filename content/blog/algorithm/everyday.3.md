-------

▶︎ 입출력 예
-------
```js

Input: [1, 2, 3, 4, 5], 2
Output: [3, 4, 5, 1, 2]

Input: [0, 1, 2, 3, 4], 1
Output: [1, 2, 3, 4, 0]
```
