## **▶︎ 문제 설명**

- 자연수 n, m의 최대 공약수와 최소 공배수를 구하시오.

## **▶︎ 입출력 예**

```js

Input : n = 2, m = 5
Output : [1, 10]

Input : n = 3, m = 7
Output : [1, 21]

```
