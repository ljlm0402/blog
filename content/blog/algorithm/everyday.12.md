문제 설명
-------

- 세 자연수 a, b, c 가 피타고라스 정리 a2 + b2 = c2 만족 시 피타고라스 수라고 합니다.
  - (여기서 a < b < c )

<br />

- a + b + c = n 인 피타고라스 수 a, b, c의 곱을 구하시오.
  - (단, 위 조건에 만족하지 않는 n은 0으로 출력하시오.)

입출력 예
-------
```sh

Input = 12
Output = 60 // 3 * 4 * 5 = 60

```
