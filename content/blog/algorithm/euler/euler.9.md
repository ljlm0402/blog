## **📕 문제 설명**

- 세 자연수 a, b, c 가 피타고라스 정리 a2 + b2 = c2 만족 시 `피타고라스 수`라고 합니다.

- a + b + c = n 인 피타고라스 수 `a, b, c의 곱`을 구하시오.

<br />

**[⬆ 목차](#-목차)**

## **🔖 제한 사항**

- 세 자연수는 a < b < c 성립되어야 합니다.

- 위 조건에 만족하지 않는 n은 0으로 출력합니다.
