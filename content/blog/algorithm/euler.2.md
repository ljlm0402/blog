## **▶︎ 문제 설명**

- 피보나치 수열의 각 항은 바로 앞의 항 두 개를 더한 것이 됩니다. 

- 짝수이면서 n 이하인 모든 항의 합을 구하시오.

## **▶︎ 입출력 예**

```js

Input : 10
Output : 10 // 2 + 8 = 10

Input : 40
Output : 44 // 2 + 8 + 34 = 44

```
