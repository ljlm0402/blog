문제 설명
-------

- n보다 작은 자연수 중에서 nums의 모든 배수들의 합을 구하시오.

입출력 예
-------
```sh

Input: n = 10, nums = [3, 5]
Output: 23 // 3 + 5 + 6 + 9 = 23

```
