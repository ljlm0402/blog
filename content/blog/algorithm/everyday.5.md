## **▶︎ 문제 설명**

- 정수 n이 주어지면, n개의 여는 괄호 "("와 n개의 닫는 괄호 ")"로 만들 수 있는 괄호 조합을 모두 구하시오.

## **▶︎ 입출력 예**

```js

Input : 1
Output : ["()"]

Input : 2
Output : ["(())", "()()"]

Input : 3
Output : ["((()))", "(()())", "()(())", "(())()", "()()()"]

```
